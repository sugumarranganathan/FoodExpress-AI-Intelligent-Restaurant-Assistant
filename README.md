# 🍽️ FoodExpress AI – Intelligent Restaurant Assistant

An AI-powered Restaurant Ordering System built using **Flask**, **Google Gemini AI**, **Google Sheets Database**, **HTML**, **CSS**, and **JavaScript**.

The system allows customers to browse the restaurant menu, place food orders, and interact with an intelligent AI chatbot that answers restaurant-related questions using restaurant knowledge.

# 🤖 AI Chatbot Features

The AI Assistant can answer questions such as:

- Burger List
- Pizza List
- Today's Offers
- Menu Details
- Food Prices
- Restaurant Information
- Opening Hours
- Delivery Information
- Contact Information

The chatbot uses Google Gemini AI together with the restaurant knowledge base to provide accurate responses.
